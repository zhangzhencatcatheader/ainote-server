## 🛠 技术栈

- **语言**: Kotlin 2.1.20
- **框架**: Spring Boot 3.5.6
- **ORM**: Jimmer 0.9.117
- **构建**: Gradle + KSP
- **数据库**: PostgreSQL/MySQL/H2
- **缓存**: Redis (可选)
- **认证**: JWT
