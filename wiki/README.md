## 🛠 技术栈

### 核心技术
- **语言**: Kotlin 2.1.20
- **框架**: Spring Boot 3.5.6
- **ORM**: Jimmer 0.9.117
- **构建**: Gradle + KSP
- **数据库**: PostgreSQL/MySQL/H2

### 集成服务
- **AI 框架**: Spring AI 1.0.0
- **通义千问**: 阿里云 Qwen 模型
- **OpenAI**: GPT 模型支持
- **短信服务**: 阿里云 PNVS
- **对象存储**: 阿里云 OSS
- **缓存**: Redis (可选)
- **认证**: JWT

### 安全特性
- **图形验证码**: Hutool Captcha
- **短信验证码**: 防刷机制
- **密码加密**: BCrypt
- **权限控制**: Spring Security

## 🌟 核心特性

### 多种认证方式
- 用户名密码登录 + 图形验证码
- 手机短信验证码登录/注册
- JWT Token 无状态认证
- 多租户选择和切换

### AI 能力
- 智能对话（支持 Qwen 和 OpenAI）
- AI 自动生成台账模板字段
- 文件理解和内容提取
- 结构化输出转换

### 企业管理
- 多企业/多租户支持
- 企业成员管理
- 角色权限体系（USER/ADMIN/COMPANYADMIN）
- 数据隔离和权限控制

### 开发体验
- 类型安全的 ORM（Jimmer）
- 自动生成 API 文档（OpenAPI/Swagger）
- TypeScript 客户端自动生成
- 操作日志自动记录
